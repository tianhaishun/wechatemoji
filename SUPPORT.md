# 获取帮助

如果你在使用 `WeChatEmoji` 时遇到问题，可以按下面的方式选择合适的入口：

## 使用问题

适合这类情况：

- 不确定如何安装或运行
- 不清楚某个功能该怎么用
- 对输出目录、导入模式或打包方式有疑问

建议先阅读：

- [README.md](README.md)
- [CONTRIBUTING.md](CONTRIBUTING.md)

如果仍然没有解决，可以提交 GitHub Issue，并尽量附上：

- 你的运行方式（源码 / EXE）
- 系统版本
- 微信版本或环境特征
- 报错截图或日志片段

## Bug 反馈

如果你认为这是程序缺陷，请使用 Bug 模板提交，并补充：

- 复现步骤
- 实际结果
- 期望结果
- 是否稳定复现

## 安全问题

如果问题涉及登录态、敏感数据、隐私残留或凭据泄露，请不要公开提交，请参考 [SECURITY.md](SECURITY.md)。
