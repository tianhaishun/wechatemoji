# 社区行为准则

为了让 `WeChatEmoji` 成为一个更友好、更可协作的项目，我们希望所有参与者都能共同维护一个尊重、包容、务实的交流环境。

## 我们鼓励的行为

- 友善沟通，尊重不同背景和经验水平的参与者
- 关注问题本身，尽量基于事实、复现步骤和证据讨论
- 给出建设性反馈，帮助彼此把事情做得更好
- 包容新手提问，减少不必要的讽刺和攻击性表达
- 在讨论安全、隐私或合规问题时保持谨慎和克制

## 不被接受的行为

- 人身攻击、羞辱、歧视、骚扰或威胁
- 恶意抬杠、反复挑衅、故意制造对立
- 未经允许公开他人的隐私、账号信息、日志或敏感数据
- 发布违法、侵权、滥用平台能力或鼓励绕过安全边界的内容
- 在已明确需要私下沟通的安全问题上继续公开扩散细节

## 适用范围

本准则适用于：

- GitHub Issues
- Pull Requests
- 代码评审评论
- 文档讨论
- 与本项目有关的其他公开协作场景

## 处理方式

如果维护者认为某些行为已经影响社区协作体验，可能会采取如下措施：

- 友善提醒
- 要求调整表达或补充事实依据
- 隐藏、锁定或关闭相关讨论
- 拒绝相关贡献
- 在必要时限制后续参与

## 如何反馈

- 一般沟通问题：可通过 Issue 或 PR 讨论提出
- 安全与敏感问题：请参考 [SECURITY.md](SECURITY.md)
- 若问题涉及公开讨论不合适的内容，请尽量私下联系仓库维护者

我们希望这个仓库既能解决实际问题，也能让参与过程本身是舒服的。感谢每一位认真交流、认真做事的人。
